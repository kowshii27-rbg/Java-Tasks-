# Java-Tasks-
Java Tasks This repository contains Java tasks demonstrating various concepts.


TASK - 1 Bank Account Management System (Java OOP)

The system supports basic banking operations such as:

Deposit money
Withdraw money
Check account balance
Interest calculation for savings accounts

It clearly illustrates:

Interface
Encapsulation
Inheritance
Polymorphism
Method Overriding

Sample Output
Deposited: 1000.0
Withdrawn: 2000.0
Balance: 4000.0
----------------

TASK - 2 Student Grade Calculator (Java)

The system performs the following operations:
Accept marks for 5 subjects
Calculate total marks
Calculate average marks
Assign grade based on average
Display result

Concepts Used:
User Input using Scanner
Arithmetic Operators
Conditional Statements (if-else)
Basic Java Program Structure
Decision Making Logic

Sample Input
Enter Marks for 5 subjects:
Enter marks for subject 1:85
Enter marks for subject 2:90
Enter marks for subject 3:78
Enter marks for subject 4:88
Enter marks for subject 5:92

Sample Output
Total Marks =433.0
Average = 86.6
Grade: A
